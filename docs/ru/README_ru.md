# ReshEge Helper

[English](../README.md) | [Русский](./README_ru.md)

ReshEge Helper — это userscript, который улучшает интерфейс игры ReshEge, добавляя современное меню, историю матчей, таблицу лидеров и возможности кастомизации.  
Дополнительно предоставляется отдельный скрипт для блокировки рекламы, который удаляет навязчивые объявления и предупреждения антиадблока на всём домене sdamgia.ru.

## ✨ Возможности

*   Удобное боковое меню (только в игре)
*   Просмотр таблицы лидеров
*   История матчей с подробной информацией об ошибках
*   Поддержка светлой и тёмной темы
*   Плавный и быстрый интерфейс
*   Индикатор состояния соединения
*   Возможность расширения функционала
*   **Сопутствующий скрипт блокировки рекламы** – скрывает рекламу и антиадблок‑уведомления на всех страницах

## 🛠️ Установка

Для использования скриптов вам потребуется установить менеджер пользовательских скриптов, например **Tampermonkey** или **Violentmonkey**.  
**Предоставляются два отдельных скрипта:** один для улучшения игры и один для удаления рекламы. Для полноценной работы рекомендуется установить оба.

### Пошаговая инструкция:

1.  **Установите менеджер пользовательских скриптов:**
    *   **Tampermonkey:** [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
    *   **Violentmonkey:** [Chrome Web Store](https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag) | [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)

2.  **Откройте панель управления менеджера.** Обычно это делается через значок расширения в правом верхнем углу браузера.

3.  **Создайте новый скрипт для игрового помощника:**
    *   В Tampermonkey нажмите `➕` (Создать новый скрипт).
    *   В Violentmonkey нажмите `+` (Создать новый скрипт).

4.  **Установите `ege_game.js`:**
    *   Откройте файл [ege_game.js](https://github.com/Danex-Exe/ege-game/blob/main/ege_game.js) в репозитории.
    *   Нажмите кнопку `Raw` для отображения исходного кода.
    *   Скопируйте весь код (Ctrl+A → Ctrl+C).
    *   Вернитесь в редактор менеджера, выделите всё содержимое (Ctrl+A) и вставьте скопированный код (Ctrl+V).
    *   Сохраните скрипт (Ctrl+S или `Save & Close`).

5.  **Создайте новый скрипт для блокировщика рекламы:**
    *   Повторите шаг 3, чтобы создать ещё один новый скрипт.

6.  **Установите `ads_cleaner.js`:**
    *   Откройте файл [ads_cleaner.js](https://github.com/Danex-Exe/ege-game/blob/main/ads_cleaner.js) в репозитории.
    *   Нажмите кнопку `Raw`, скопируйте весь код.
    *   Вставьте его в редактор нового скрипта и сохраните.

7.  **Проверьте установку:**
    *   Перейдите на страницу игры [«Держи оборону»](https://ege.sdamgia.ru/game.htm).
    *   В левой части экрана должна появиться кнопка **«МЕНЮ»** (работа `ege_game.js`).
    *   Откройте любую другую страницу сайта (например, каталог или тест) — реклама и окно «Отключите блокировщик рекламы» должны исчезнуть (работа `ads_cleaner.js`).
    *   Готово! Вы успешно установили оба компонента ReshEge Helper.

## 📖 Использование

### Игровой помощник (`ege_game.js`)

После установки в интерфейсе **только на странице игры** появится кнопка **«МЕНЮ»**. Нажмите на неё, чтобы открыть боковую панель.

Через панель можно:

*   **Таблица лидеров:** Просмотр лучших игроков.
*   **История:** Детальный разбор прошедших матчей.
*   **Настройки:** Переключение между светлой и тёмной темой оформления.

### Блокировщик рекламы (`ads_cleaner.js`)

Этот скрипт запускается автоматически на всех страницах домена `sdamgia.ru`. Он скрывает:

*   Рекламные блоки Яндекс.Директ
*   Всплывающие предупреждения антиадблока
*   Другие рекламные элементы

Никакой настройки не требуется — скрипт работает незаметно в фоновом режиме.

## 📜 История матчей

История матчей отображает информацию о прошедших играх: соперников, изменение рейтинга и время. Также доступен подробный разбор ошибок по каждому заданию.

## 🎨 Темы

Доступны светлая и тёмная темы. Выбранная тема сохраняется автоматически.

## ⚙️ Архитектура

Проект построен вокруг модульной системы управления меню, которая отвечает за отображение интерфейса и навигацию. Данные загружаются динамически и отображаются в UI.  
Блокировщик рекламы представляет собой легковесный самостоятельный скрипт, который перехватывает рекламные запросы и скрывает определённые элементы DOM, не мешая работе игрового помощника.

## 🧩 Расширение

Скрипт можно расширять, добавляя новые элементы и функции без изменения основной логики.  
Блокировщик рекламы можно настраивать, редактируя список CSS‑селекторов.

## 🐛 Отладка

Информация о работе скрипта доступна в консоли браузера (F12 → Console).
